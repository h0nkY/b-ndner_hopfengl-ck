# Hopfenglück - Handgebrautes Bier aus den Schweizer Alpen

Eine moderne, hochwertige One-Page Website für die Craft Beer Marke "Hopfenglück".

## 🎨 Design & Stil

- **Visueller Stil**: Premium, rustikal, alpin, handgefertigt
- **Inspiration**: Schweizer Berge, traditionelles Brauen, warme erdige Töne
- **Farben**: Beige, Braun, Gold, Dunkelgrün
- **Typografie**: Elegante Serif für Überschriften, saubere Sans-Serif für Fliesstext
- **Stimmung**: Authentisch, lokal, freundlich, hochwertig

## 📁 Projektstruktur

```
windsurf-project/
├── index.html          # Haupt-HTML-Datei
├── styles.css          # CSS-Stile
├── script.js           # JavaScript-Funktionalität
├── assets/             # Bilder und Grafiken
│   ├── logo.svg        # Haupt-Logo
│   ├── logo-large.svg  # Grosses Logo für Hero-Sektion
│   ├── alps-mountains.jpg # Hintergrund für Hero-Sektion
│   ├── alps-sunset.jpg  # Hintergrund für Experience-Sektion
│   ├── brewery.jpg     # Brauerei-Bild
│   └── beer-bottle.png # Bierflasche
├── README.md           # Diese Datei
└── package.json        # Projekt-Metadaten
```

## 🚀 Features

### ✅ Implementierte Funktionen

- **Responsive Design**: Mobile-first Ansatz für alle Geräte
- **Smooth Scrolling**: Sanftes Scrollen zwischen Sektionen
- **Sticky Navigation**: Feste Navigationsleiste mit Scroll-Effekten
- **Animationen**: Fade-in Effekte und Micro-Interactions
- **Altersverifikation**: Optionaler 18+ Popup
- **Kontaktformular**: Funktionales Formular mit Validierung
- **SEO-freundlich**: Semantische HTML5-Struktur
- **Performance**: Lazy Loading für Bilder, optimierte Animationen

### 📱 Sektionen

1. **Hero Section**: Vollbreite Hintergrund mit Call-to-Action
2. **About/Story**: Geschichte und Philosophie der Marke
3. **Product Section**: Produktvorstellung mit Fakten
4. **Experience**: Emotionale Markenerfahrung
5. **Availability**: Wo erhältlich
6. **Contact**: Kontaktinformationen und Formular
7. **Footer**: Links und Copyright

## 🎯 Anpassung

### Platzhalter-Texte ersetzen

Die folgenden Platzhalter können einfach ersetzt werden:

- `index.html`: Alle Textinhalte und Überschriften
- `styles.css`: Farben und Abstände anpassen
- `assets/`: Eigene Bilder hinzufügen

### Farben anpassen

Die Farbvariablen in `styles.css` können einfach angepasst werden:

```css
:root {
    --color-beige: #F5E6D3;
    --color-brown: #8B4513;
    --color-gold: #D4AF37;
    --color-dark-green: #2F4F2F;
    /* ... */
}
```
